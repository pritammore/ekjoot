Add blog posts in read me later lists using Ajax.

== Description ==

This plugin will allow user to save their favourite blog post in a certain area so that they can read them later.

This feature is useful for magazine style blogs which provides a lot of content everyday. 

We make this system so that logged in users can save interesting posts in member’s only area so that they can come back and read them later.

== Installation ==

Go to the WordPress administration area then on PLUGINS , ADD NEW , UPLOAD and browse to  the zip file you downloaded  and click INSTALL.
Activate  through the 'Plugins' menu in WordPress.

= Widget =

You should use the "Read Me Later" widget on your pages.

After you have installed the plugin follow the bellow steps:

1. Add "Read Me Later" widget to the sidebar of your choosing.
2. Then click "Read Me Later" button from the bottom of your favourite post to save them.
